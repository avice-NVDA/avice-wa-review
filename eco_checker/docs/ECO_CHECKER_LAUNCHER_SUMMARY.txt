TECHNICAL DETAILS
-----------------

Launcher Architecture:
- Script location detection: Uses $0 and dirname/pwd
- Python path: /home/utils/Python/builds/3.11.9-20250715/bin/python3
- Argument passing: $argv:q (preserves quoting and special characters)
- Exit code propagation: Uses $status
- Error handling: Checks for Python and eco_checker.py existence

C-Shell Features Used:
- #!/bin/csh -f (no .cshrc sourcing for speed)
- set variables for paths
- if/then for error checking
- exit with preserved status code
