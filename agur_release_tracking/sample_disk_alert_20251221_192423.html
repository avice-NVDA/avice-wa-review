Quick Disk Analysis Commands (CSH/TCSH Compatible)

Copy-paste these commands to analyze disk usage. Run from workarea root directory.

Find top 10 largest directories: 
du -sk * |& grep -v '^du:' | sort -rn | head -10

Shows size in KB for each directory, sorted largest first

Count and size of PnR work directories: 
find pnr_flow/*/work_* -maxdepth 0 -type d |& wc -l
find pnr_flow/*/work_* -maxdepth 0 -type d -exec du -sk {} \; |& sort -rn | head -5

First line: total count. Second line: top 5 largest work dirs

Count and size of PT timing sessions: 
find signoff_flow/auto_pt/work_* -maxdepth 0 -type d |& wc -l
find signoff_flow/auto_pt/work_* -maxdepth 0 -type d -exec du -sk {} \; |& sort -rn | head -5

First line: total count. Second line: top 5 largest PT sessions

Find large log files (>100MB): 
find . -name "*.log" -type f -size +102400 -exec ls -lh {} \; |& awk '{print $5, $9}' | head -10

Shows file size and path for logs >100MB (102400 blocks = 100MB)

Find old work directories (>60 days): 
find pnr_flow/*/work_* signoff_flow/*/work_* -maxdepth 0 -type d -mtime +60 |& grep -v '^find:'

Lists all work directories older than 60 days (prime cleanup candidates)

Star SPEF files disk usage: 
find signoff_flow/star/work_*/spef -type f -name "*.spef.gz" -exec du -sk {} \; |& sort -rn | head -10

Shows largest SPEF files in KB (old corners can often be deleted)

  • sort -rn = reverse numeric sort (largest first)
